# Landing Page: Swim Schools

## What This Is
A landing page targeting swim school owners as potential ConvoPro customers. URL: `getconvopro.com/swim-schools`

## Source File
`swim-schools.html` - complete HTML with inline CSS, ready for WordPress.

---

## Instructions for Adding to WordPress

### Step 1: Create the Page
1. In WordPress admin, go to **Pages → Add New**
2. Set the title to "Swim Schools" (or "AI for Swim Schools")
3. Set the URL slug to `swim-schools`
4. Choose a **blank/full-width template** if available (no sidebar)

### Step 2: Add the Content
1. In the block editor, click **+** and add a **Custom HTML** block
2. Open `swim-schools.html` and copy everything from `<style>` through `</body>` (skip the `<!DOCTYPE>`, `<html>`, and `<head>` tags)
3. Paste into the Custom HTML block

### Step 3: SEO Settings (if using Yoast or similar)
- **SEO Title:** AI Answering Service for Swim Schools | ConvoPro
- **Meta Description:** Never miss another enrollment call. ConvoPro answers parent inquiries 24/7 via phone, text, and chat—in English and Spanish.
- **Focus Keyphrase:** AI answering service swim schools

### Step 4: Publish and Verify
1. Click **Publish**
2. Visit `getconvopro.com/swim-schools` to verify it looks correct
3. Test on mobile
4. Submit URL to Google Search Console for indexing

## Customization Notes

### To change the phone number:
Search for `+12816990999` and `(281) 699-0999` — replace both with the correct number.

### To add a logo:
Add an `<img>` tag in the hero section above the `<h1>`.

### To change colors:
- Primary blue: `#3b82f6`
- Dark background: `#0f172a`
- Error/pain red: `#ef4444`
- Success green: `#059669`

### To add testimonials later:
Insert a new `<section>` between Features and CTA with customer quotes.
